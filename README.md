

## Installation

```bash
$ npm install
$ npm build
```
```
./ngrok http 3000
$ npm build
```

## Running the app

```bash
# bind ngrok to port
./ngrok http 3000

# start with watch mode
$ npm run start:dev

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
